Home.js -

Home.js는 React Native 애플리케이션의 홈 화면을 나타내는 컴포넌트입니다. 
이 화면은 앱이 처음 실행될 때 사용자에게 표시되는 기본 화면입니다. 
Home 컴포넌트는 다음과 같은 주요 기능을 가지고 있습니다:

Home.js는 홈 화면을 구성하고, 
사용자가 다른 화면으로 이동할 수 있는 버튼 기능을 제공합니다.

MenuDetail.js -

 MenuDetail 컴포넌트는 이 props들을 이용하여 메뉴의 상세 정보를 화면에 출력합니다. 
이때 사용하는 것이 route.params.menu라는 객체입니다.
 이 객체는 메뉴의 이름(name), 아이디(id), 가격(price), 설명(description)을 담고 있습니다. 
이 정보들을 텍스트로 출력하여 메뉴의 상세 정보를 사용자에게 보여줍니다.

 React Navigation은 React Native 애플리케이션에서 탐색을 관리하기 위한 네비게이션 라이브러리입니다.

stacks.js - 

"Stack"은 React Navigation 라이브러리에서 사용되는 용어 중 하나입니다. 
"Stack" 네비게이션은 사용자가 앱의 화면 간을 네비게이트할 수 있는 방법을 제공합니다.
"Stack"은 일련의 화면을 스택 형태로 구성합니다. 
즉, 새로운 화면이 추가될 때마다 이전 화면 위에 새 화면이 쌓입니다. 
사용자가 뒤로 가기 버튼을 누르면 스택의 최상단 화면이 제거되어 이전 화면으로 돌아갑니다. 
이는 사용자가 앱 내에서 이동하는 경로를 정확하게 따라가는 방식이며, 
많은 모바일 앱에서 이러한 유형의 네비게이션을 사용합니다.
예를 들어, 사용자가 "Home" 화면에서 "Details" 화면으로 이동하면, 
"Details" 화면이 "Home" 화면 위에 쌓입니다. 사용자가 뒤로 가기를 선택하면, "Details" 화면이 스택에서 제거되고, 
사용자는 다시 "Home" 화면으로 돌아갑니다.
React Navigation 라이브러리는 이러한 "Stack" 네비게이션을 구현하는 데 필요한 도구와 컴포넌트를 제공합니다.
 이 라이브러리를 사용하여 앱의 화면 간 이동을 관리할 수 있습니다. 
 이를 통해 사용자에게 일관된 및 직관적인 네비게이션 경험을 제공할 수 있습니다.

HomeStacks.js - 

HomeStacks는 React Navigation 스택 내비게이터의 인스턴스입니다. 이 인스턴스는 앱에서 화면 전환을 관리합니다.
스택 내비게이션은 사용자가 앱 내에서 화면을 전환하는 방식을 관리하는 시스템입니다. 
이는 앱 내에서 화면을 전환하는 방식을 관리하는 시스템입니다. 
이는 일련의 화면들을 스택(stack) 형태로 관리하는 방식으로, 
사용자가 새로운 화면으로 이동할 때마다 그 화면이 스택의 맨 위에 쌓이게 됩니다. 
사용자가 뒤로 가기를 선택하면 현재 화면이 스택에서 제거되고 이전 화면이 표시됩니다.
HomeStacks 함수 내부에서는 Stack.Navigator와 
Stack.Screen 컴포넌트를 사용하여 스택 내비게이터를 설정하고 각 화면을 정의합니다.
이는 앱의 메인 화면이며, 
여기서 사용자는 다른 화면으로 이동할 수 있습니다.
HomeStacks의 역할은 앱 내에서 화면 전환을 관리하고 각 화면의 설정을 제어하는 것입니다. 
이 함수는 앱의 구조를 정의하고 사용자가 앱에서 화면을 전환하는 방식을 관리합니다.
이 함수는 React Native 앱에서 호출되며, 반환 값은 앱의 메인 화면을 구성하는 JSX입니다. 

App.js - App.js는 React와 React Native 앱에서의 진입점(Entry Point) 파일로써, 앱이 시작될 때 가장 먼저 실행되는 스크립트입니다. 
이 파일에서 앱의 주요 부분을 구성하며, 여러 페이지 또는 기능을 가져와서 전체 앱의 구조를 만들어 냅니다.
● : 필요한 모듈 불러오기  ● : 앱 컴포넌트 정의  ● : 렌더링  ● : 스타일링  ● : 네비게이션

App.js는 주로 앱의 전반적인 구조와 로직을 정의하며, 
하위 컴포넌트들을 조합해 앱의 전체적인 레이아웃을 만드는 역할을 합니다. 
이 파일은 앱의 가장 중요한 부분이기 때문에 이해하고, 잘 관리하는 것이 중요합니다.
